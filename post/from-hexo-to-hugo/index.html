Hexo迁移至Hugo

@Vozhuo · Mar 9, 2019 · 1 min read

本站使用的静态博客框架已从Hexo迁移至Hugo。Hugo生成博客的速度要远大于Hexo，不过主题没有Hexo多。我选择的主题是Hugo Nuo。

Hugo没有像Hexo那样专门的部署命令deploy，所以为了方便，需要自己写一个部署脚本。

@echo off
echo 'Generating...'
hugo
cd public
git add -A
git commit -m 'Updating'

echo 'Deploying...'
git push

echo 'Done'

cd ..
